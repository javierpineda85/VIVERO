Este sistema fue creado mientras realizaba la tecnicatura como analista y programador de sistemas.

Al recuperar el HDD que contenia los archivos de este proyecto pude recurar una versión que no es estable.
Es decir que contiene errores y falta depurar detalles que ya estaban corregidos.

Está realizado en Visual Basic 6.0 con conexión a una base de datos de Acces mediante senticia de codigo SQL.

Representa uno de los desafios más grandes que tuve como programador ya que tenia que satisfacer las necesidades de la empresa, 
inclusive con información que todavia no conocia. Tuve que pasar muchas horas buscando en la web lo que mi cliente me pedia.

Su arquitectura fue creada para cierto fin que luego fue escalando y se tuvo que ir adaptando el sistema a los nuevos desaíos.

Todavia años despues de realizado, se que tiene muchas cosas para corregir pero siempre será el mi primer sistema grande para una empresa.

Este sistema cuenta con: 
* AMB de clientes, proveedores, usuarios, y personal de la empresa
* Ingresos por usuarios con niveles
* Control de cheques propios y de terceros
* Visualizacion de cheques en cartera e ingresados al banco
* Control de flujo de caja
* Seguimiento de cuentas corrientes
* Impresiones de remitos para control interno
